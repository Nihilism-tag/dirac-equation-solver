# 狄拉克方程数值求解器工程说明

## 项目概述

本项目实现了一套用于求解含屏蔽库仑势的单电子径向狄拉克方程的数值计算框架。主要包含三种数值方法：**B-样条法（B-spline）**、**映射代数基伪谱法（MAB-GPS）**以及基于MAB-GPS的**稳定化方法（Stabilization Method）**，用于在正能连续区中识别共振态。

### 核心物理问题

求解以下径向狄拉克方程：

$$
\hat{H} = \begin{pmatrix} V(r) + mc^2 & c(-\frac{d}{dr} + \frac{\kappa}{r}) \\ c(\frac{d}{dr} + \frac{\kappa}{r}) & V(r) - mc^2 \end{pmatrix}
$$

其中：
- **势能项**：$V(r) = -\frac{Z}{r}e^{-\lambda r}$（带屏蔽的库仑势）
- **核电荷数**：$Z=92$（类铀体系）
- **屏蔽参数**：$\lambda=0.1$
- **量子数**：$\kappa=1$（对应$p_{1/2}$态，为共振搜索提供离心势垒）

## 数值方法策略与原理

### 1. MAB-GPS方法（核心求解器）

#### 1.1 节点选择策略：切比雪夫-高斯-洛巴托（CGL）节点

**选择理由**：
- **原代码缺陷**：使用Legendre-Gauss节点（开区间$(-1,1)$），无法覆盖原点$r=0$
- **CGL优势**：包含端点$x=-1$和$x=+1$，经映射$r(x) = L\frac{1+x}{1-x+\alpha}$后可精确到达物理边界
- **解析公式**：$x_k = -\cos\left(\frac{k\pi}{N-1}\right)$，$k=0,...,N-1$

**关键代码**：
```python
def _cgl_nodes(N):
    k = np.arange(N)
    return -np.cos(np.pi * k / (N - 1))  # 包含±1端点
```

#### 1.2 微分矩阵构建：Trefethen谱微分公式

**原理**：基于重心拉格朗日插值的谱微分矩阵

$$
D_{ij} = \frac{c_i}{c_j} \frac{(-1)^{i+j}}{x_i - x_j} \quad (i \neq j)
$$

**对角元约束**（关键稳定性保障）：
$$
D_{ii} = -\sum_{j \neq i} D_{ij}
$$

其中$c_k=2$（端点）或$1$（内部）。该求和规则确保微分算子的守恒性。

**选择理由**：
- 谱精度：指数收敛而非多项式收敛
- 边界聚集：CGL节点在边界处密集，适合解析边界层行为

#### 1.3 边界条件实施：矩阵截断法

**策略**：提取内部节点（排除首尾端点）

```python
interior_idx = np.arange(1, N - 1)  # 索引1到N-2
r_interior = r_all[interior_idx]   # 不含r=0和r=r_max
D_r_interior = D_r_all[np.ix_(interior_idx, interior_idx)]
```

**物理意义**：
- 强制$P(0)=Q(0)=0$（原点正则性）
- 强制$P(r_{max})=Q(r_{max})=0$（边界硬壁）
- 结果：$r_{min} = 1.06 \times 10^{-4}$ a.u.（避开奇点）

#### 1.4 哈密顿矩阵求解：非对称实部提取法

**历史演进**：

| 阶段 | 策略 | 问题 | 当前方案 |
|------|------|------|----------|
| 原始版 | `eig`+虚部过滤 | 非厄密、40%特征值丢弃 | ❌ 废弃 |
| 修复版 | 对称化+`eigh` | 奇偶解耦、Nyquist振荡 | ❌ 废弃 |
| **当前版** | `eig`+`.real` | 保持谱精度、无虚部过滤 | ✅ 采用 |

**当前策略原理**：
```python
eigenvalues_complex, eigenvectors = linalg.eig(H)
eigenvalues = eigenvalues_complex.real  # 虚部~1e-14数值噪声
```

**选择理由**：
1. **保持谱精度**：避免对称化导致的中央差分退化
2. **无奇偶解耦**：Nyquist模式($+1,-1,+1,-1...$)不会出现
3. **波函数平滑**：验证通过`oscillation_rate < 0.5`测试
4. **全谱保留**：296个特征值全部保留（$2 \times (150-2)$）

### 2. 稳定化方法（共振态搜索）

#### 2.1 三阶段流程

```
Phase 1: L尺度扫描 (L ∈ [0.5, 3.0], step=0.05)
    ↓ 收集能隙内本征值
Phase 2: 能级跟踪与共振识别
    ↓ 计算dE/dL，寻找平台
Phase 3: 可视化输出
```

#### 2.2 L尺度扫描原理

**物理机制**：改变映射参数$L$相当于改变计算"盒子"的大小
- 连续态能量：随$L$增加而下降（$-1/L^2$趋势）
- 共振态能量：几乎不随$L$变化（平台特征）

**能量窗口**：$[0.9mc^2, 1.2mc^2]$（正能连续区）

#### 2.3 能级跟踪算法

**贪心最近邻匹配**：
```python
# 对相邻L步的能量本征值进行匹配
for each new_eigenvalue:
    find nearest unassigned track within threshold (0.5 * mc2 * L_step)
    if matched: assign to existing track
    else: start new track
```

**NaN填充策略**：处理变长本征值集合，保证轨迹矩阵规整

#### 2.4 共振识别准则

**平台判据**：在中心60%的$L$范围内，$|dE/dL|$方差最小

```python
# 评分函数
score = var(|dE/dL|) over central 60% range
resonance = argmin(score)
```

**结果示例**（实际运行）：
- 共振能量：$E_{res} = 0.999892 \, mc^2$（18776.84 a.u.）
- 最优尺度：$L_{opt} = 1.0000$

### 3. CCR复能量扫描方法（共振谱分析）

#### 3.1 物理机制

**CCR（Complex Coordinate Rotation）方法**通过在复平面上旋转坐标来识别共振态。与稳定化方法不同，CCR直接在复能量平面上搜索共振结构，而非依赖L尺度的平台特征。

**关键思想**：
- 引入复旋转参数$\theta$（旋转角）
- 对每个$\theta$值求解完整的复能量本征值谱
- 共振态在复平面上形成特征轨迹（随$\theta$变化）
- 连续态轨迹与共振态轨迹可通过形态学特征区分

#### 3.2 数据采集流程

**第一步：theta扫描**（`ccr_scan.py`）

```bash
python ccr_scan.py --theta-start 0.0 --theta-stop 1.57 --theta-step 0.01 --out ccr_data.npz
```

参数说明：
- `--theta-start`, `--theta-stop`, `--theta-step`：theta角度范围与步长（弧度）
- `--out`：输出`.npz`文件路径（必需）
- `--N`：CGL节点数（默认读取`config.py`的`N_POINTS`）
- `--L`：径向映射尺度（默认读取`config.py`的`L_SCALE`）
- `--alpha-map`：映射参数（默认读取`config.py`的`ALPHA_MAP`）

**输出`.npz`文件结构**：

| 键名 | 形状 | 数据类型 | 说明 |
|------|------|----------|------|
| `E_re` | `(n_theta, n_eig)` | float64 | 实部能量矩阵，单位a.u. |
| `E_im` | `(n_theta, n_eig)` | float64 | 虚部能量矩阵，单位a.u. |
| `thetas` | `(n_theta,)` | float64 | theta值数组（弧度） |
| `mc2` | 标量 | float64 | 电子静止质量能量（用于归一化） |
| `meta_Z` | 标量 | float64 | 核电荷数 |
| `meta_kappa` | 标量 | float64 | 量子数$\kappa$ |
| `meta_lambda` | 标量 | float64 | 屏蔽参数$\lambda$ |
| `meta_N` | 标量 | float64 | CGL节点数 |
| `meta_L` | 标量 | float64 | 径向映射尺度 |
| `meta_alpha` | 标量 | float64 | 映射参数$\alpha$ |
| `meta_c` | 标量 | float64 | 光速（a.u.） |

**为何采用`.npz`格式**：
- 支持2D数组（`E_re`, `E_im`）的高效存储
- 保留双精度浮点精度（float64）
- 元数据与数据一体化存储，便于重现性
- 相比CSV避免了文本转换的精度损失

## 重要技术决策记录

### 决策1：放弃对称化，采用`eig+.real`

**背景**：H_issue.md发现对称化`H=0.5*(H+H.T)`导致奇偶解耦（Nyquist振荡）

**分析**：
- 对称化将全局谱算子退化为局部中央差分
- Nyquist模式（$+1,-1,+1,-1...$）的导数为零，形成"零动能陷阱"
- 求解器利用此陷阱叠加高频噪声以降低总能量

**决策**：移除对称化，改用`scipy.linalg.eig`+`.real`提取

**验证**：波函数平滑性测试`oscillation_rate=0.163 < 0.5` ✅

### 决策2：CGL节点+矩阵截断实施边界条件

**对比方案**：
- Plan A（采用）：CGL节点+截断，标准本征值问题
- Plan B（放弃）：Galerkin方法+广义本征值问题（$Hc=ESc$）

**选择理由**：
- Plan A在保持谱精度的同时强制边界条件
- Plan B需构造重叠矩阵$S$，增加复杂度
- Plan A与B-样条法形成清晰对比（标准vs广义本征值）

### 决策3：稳定化方法参数选择

**L扫描范围**：[0.5, 3.0]，步长0.05 → 51个L值

**理由**：
- $L<0.5$：盒子过小，共振态受边界干扰
- $L>3.0$：连续态过于密集，跟踪困难
- 步长0.05：平衡分辨率与计算成本

**能量窗口**：[0.9, 1.2] $mc^2$

**理由**：
- 避开$mc^2$以下的束缚态（28个物理态）
- 聚焦正能连续区的共振结构

## 缺陷修复历程

### Defect 1：$r_{max}$过小导致的假通过

**现象**：能量误差<1e-6但波函数在$r=1.0$ a.u.处截断

**根因**：$L=0.1, \alpha=0.2$ → $r_{max}=1.0$ a.u.，铀原子1s态需>7 a.u.才能衰减

**修复**：$L=1.0, \alpha=0.1$ → $r_{max}=20$ a.u.

**验证**：$P(r_{max})=6.40 \times 10^{-06} \ll 0.01$ ✅

### Defect 2：非厄密哈密顿矩阵

**现象**：40%本征值因虚部>1e-8被丢弃

**根因**：Legendre-Gauss节点+直接构造导数矩阵 → 非对称

**修复**：CGL节点+对称化（历史方案）→ 最终改为`eig+.real`

**验证**：296个本征值全部实数保留 ✅

### Defect 3：原点无边界条件

**现象**：波函数在原点处垂直跳变

**根因**：开区间节点不含$r=0$，无法强制$P(0)=0$

**修复**：CGL节点含端点，矩阵截断排除$r=0$和$r=r_{max}$

**验证**：$r_{min}=1.06 \times 10^{-04} > 0$ ✅
